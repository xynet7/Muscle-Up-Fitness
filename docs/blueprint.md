# **App Name**: MuscleUp

## Core Features:

- Membership Plan Display: Showcase various gym membership plans with details (duration, price, benefits) on the user-facing panel.
- Subscription Purchase: Enable users to select and purchase a membership plan, generating a unique transaction receipt.
- Receipt Generation: Automatically generate unique, printable receipts upon successful membership purchase, including transaction details.
- Admin Login: Secure login panel for gym administrators with role-based access control.
- Membership Management: Admin tool for managing all active and expired subscriptions.
- Plan Management: Admin tool to create, edit, and remove gym membership plans.
- Personalized Workout Plan Suggestions: AI tool suggesting a personalized workout plan based on user fitness goals and subscription level.

## Style Guidelines:

- Primary color: Deep crimson (#DC143C) to convey intensity and power, relating to physical exertion.
- Background color: Off-white (#F2F0F1), providing a clean and contrasting backdrop that emphasizes boldness and allows the colors and design elements to stand out effectively.
- Accent color: Dark gray (#4A4A4A) to represent stability and solidity.
- Headline font: 'Space Grotesk' sans-serif for headlines; body font: 'Inter' sans-serif for body.
- Use strong, vector-based icons to represent different membership features and gym facilities.
- A dynamic, grid-based layout to present plans and information in a structured manner.
- Subtle transitions and hover effects to guide the user and add interactivity.
